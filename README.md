# Paper Storyteller

Narrative-driven academic paper writing for ML/CV/NLP. Goes beyond structural correctness to build story arcs — the kind that Wu, Efros, Liu, Freeman, and Isola write.

## When to Use

- Drafting or revising any paper section (abstract, intro, related work, method, experiments, conclusion)
- Existing intro feels generic ("Recent advances in X...")
- Paper reads like a tech report, not a story

## Usage

```
/paper-storyteller
```

Works on the current paper in your project. Reads existing sections, suggests narrative improvements.

## Install

```bash
ln -sf $(pwd) ~/.claude/skills/paper-storyteller
```
